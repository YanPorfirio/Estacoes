# Estações do Ano

Site estático sobre as estações do ano, com páginas dedicadas a verão, outono, inverno e primavera.

## Estrutura

- `index.html`: página inicial.
- `pages/`: páginas internas de cada estação.
- `assets/css/`: estilos globais.
- `assets/js/`: interações do menu e animações de entrada.
- `assets/img/`: imagens originais.
- `assets/img/optimized/`: imagens usadas pelo site, redimensionadas e recomprimidas.
- `assets/icons/`: favicon e ícones locais.

## Como executar

Abra `index.html` no navegador. Como o projeto é estático, não há etapa de build.

## Observações

As imagens originais foram mantidas em `assets/img/` para preservação. O site usa as versões otimizadas de `assets/img/optimized/`.
